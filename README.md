# yunjian.repo
### 启动过程
* npm install dva-cli -g
* dva -v
* 确保版本是0.7.0 或以上
* npm install 
* npm start 启动服务 (已经添加了用户列表部分  http://localhost:8000/#/users)
* npm run build 构建应用
